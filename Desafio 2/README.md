🧮 Calculadora de Partidas Rankeadas

Projeto desenvolvido em JavaScript para calcular o saldo de vitórias e classificar o nível de um jogador.

🚀 Funcionalidades
Cálculo de saldo de vitórias
Classificação por níveis:
Ferro
Bronze
Prata
Ouro
Diamante
Lendário
Imortal
🛠️ Tecnologias
JavaScript
▶️ Como executar
node src/index.js
📊 Exemplo de saída
O Herói tem saldo de 10 vitórias e está no nível Bronze
